1. go help
دستور go help برای نمایش اطلاعات راهنما در مورد دستورات مختلف زبان Go و ابزارهای مرتبط با آن استفاده می‌شود. با استفاده از این دستور می‌توانید کمک و مستندات مختلف در مورد ابزارها و دستورات Go را مشاهده کنید.

نحوه استفاده:

go help


2. go build a.go
دستور go build برای کامپایل کردن کد Go و تولید فایل اجرایی استفاده می‌شود. با استفاده از این دستور می‌توانید فایل‌های Go خود را کامپایل کرده و برنامه‌ای اجرایی بسازید.

نحوه استفاده:

go build a.go

این دستور فایل a.go را کامپایل کرده و یک فایل اجرایی به نام a (در سیستم‌های لینوکسی/مک) یا a.exe (در ویندوز) ایجاد می‌کند.

اگر نام فایل ورودی را ندهید، Go به طور پیش‌فرض سعی می‌کند فایل‌های .go را در پوشه جاری پیدا کرده و کامپایل کند.

برای اجرای فایل ساخته‌شده، از دستور زیر استفاده می‌کنید:

./a   # در لینوکس/مک
.\a.exe  # در ویندوز

go help: برای دسترسی به راهنمای دستورات مختلف.
go build a.go: برای کامپایل کد Go و ساخت برنامه اجرایی.
go clean: برای پاک‌سازی فایل‌های ساخته‌شده.
go doc: برای مشاهده مستندات مربوط به پکیج‌ها، توابع و انواع Go
go run a.go
go mod init a
